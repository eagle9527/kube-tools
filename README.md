# kube-tools
```
kube file browser tools

Usage:
  kube-tools [command]

Available Commands:
  cat         Create the DIRECTORY(ies), if they do not already exist.
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  ls          List information about the FILEs (the current directory by default).
  mk          Create the DIRECTORY(ies), if they do not already exist.
  mv          Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.
  rm          Remove (unlink) the FILE(s).
  tar         The default action is to add or replace tar file entries from list.
  touch       Get content from standard input and write it to file.
  version     Print the version number of kube-tools.

Flags:
  -h, --help   help for kube-tools

Use "kube-tools [command] --help" for more information about a command.
```
